# **App Name**: Sheet Formatter

## Core Features:

- Form Input: Input fields corresponding to columns in the 'Products to Monitor' sheet to add data entries.
- Sheets Connection: Allow inputting the Google Sheets URL. Retain user data through local storage in case they come back to the same sheet later.
- Form Validation: Form validation based on data types from Google Sheets columns.
- Easy data submission: A clean, accessible data submission user interface.

## Style Guidelines:

- Primary color: Light cyan (#B0E2FF) to bring a calm feeling to data entry.
- Background color: Off-white (#F5F5F5), a light shade of the primary color.
- Accent color: Soft lavender (#E6E6FA) to provide visual distinction without being too distracting.
- Font pairing: 'Inter' sans-serif for both body text and headings for clean UI
- Simple, single-column form layout with clear labels and tooltips to minimize user error.
- Use simple icons for input field hints
- Subtle animation for user interaction (e.g., form submission, validation).
